## ğŸš€ STARTUP GUIDE

### **1. Setup Flutter on your Mac**
Ensure Flutter is installed and in your PATH. If not:
1.  **Download Flutter SDK**: [flutter.dev/docs/get-started/install/macos](https://docs.flutter.dev/get-started/install/macos)
2.  **Add to PATH**: `export PATH="$PATH:[path_to_flutter]/bin"`

### **2. Run the App**
1.  Open **Android Studio**.
2.  Open the `flutter_sahaayak` folder.
3.  Ensure your **Android Emulator** is running.
4.  Run `flutter pub get` in the terminal.
5.  Click **Run** (Green Arrow) to launch.
